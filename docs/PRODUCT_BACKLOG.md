# Backlog — עדיפויות מוצר (מאגר, CRM, תורה, וואטסאפ)

מסמך ארגון בלבד; לא מקור אמת טכני. עודכן עם תוכנית באגים / ספרי תורה.

## קרוב לביצוע (אחרי ייצוב תיקוני UI)

| אזור | פריט | הערות |
|------|------|--------|
| מאגר | שיתוף לוואטסאפ/מייל + טמפלט הצעת מחיר | קיים חלקית (`lib/market/shareOfferText`, קישורים ב-`MarketClient`) |
| וואטסאפ | תבנית חכמה לפני שידור | להרחיב `BroadcastTab` |
| תנועות | תזכורות / הערות / AI | `app/transactions` |
| תורה | טפסי יצירת שקית עם `qa_kind`, `cost_amount`, `report_url`, `vendor_label` | DB ב-`067_torah_qa_batch_extensions.sql`; UI להשלים |

## בינוני טווח

| אזור | פריט |
|------|------|
| מייל | ייבוא היסטוריה לאיש קשר, סיכום הקשר, סנכרון דו-צדדי |
| CRM | כתובת/עיר לכל איש קשר, אמצעי קשר דומיננטי, מיזוג ידני, מספר טלפונים |
| CRM | סנכרון אנשי קשר מוואטסאפ / טלפון כשר |
| סופרים | גלריה בלי הגבלה + סנכרון |
| סופרים | תגית קהילה (רשימה + הוספה) — ראו `053_sofer_community` |
| תורה | מחולל חוזים + מדידת רווחיות מתוכננת מול ביצוע |
| לוח שנה | עברי, אזורים אישיים (לקוח / סופר), אבטחה |
| שידורים | תזמון כשלא מחובר | קיים `065_broadcast_schedule` — להרחיב UX |
